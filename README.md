## 使用方法
1. 编译运行简单图的广度优先算法
```shell
g++ breadth_first_search.cpp
./a.out
```
2. 编译运行栅格图的广度优先算法
```shell
g++ breadth_first_search_2.cpp
./a.out
```
3. 编译运行找到目标点就退出的广度优先算法
```shell
g++ early_exit.cpp
./a.out
```
4. 编译运行找到目标点就退出的dijkstra算法
```shell
g++ dijkstra.cpp 
./a.out
```
